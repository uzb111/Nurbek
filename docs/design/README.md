# AgroTahlil — ikki ekranli UI konsepsiyasi

Interfeys rahbariyat va amaliy GIS ishlari uchun ikki aniq vazifaga ajratildi:

1. **Umumiy tahlil** — tuman suv balansi, rasmiy limit, real ET, ob-havo, ekin va ma’lumot sifati ko‘rsatkichlari. Xarita yo‘q.
2. **Dalalar xaritasi** — dala qidiruvi, ekin kiritish, tavsiya, split, dala pasporti hamda kanal–quloq suv yo‘li line charti.

## Dizayn fayllari

- `premium-dashboard-concept.png` — rahbariyat dashboardi uchun generativ premium konsept.
- `premium-field-map-concept.png` — dala xaritasi va tahlil ekrani uchun generativ premium konsept.
- `implemented-dashboard.png` — amaldagi koddan olingan tekshiruv rasmi.
- `implemented-map.png` — amaldagi xarita ekranidan olingan tekshiruv rasmi.

## Asosiy UI qoidalari

- Bir ekranda bitta asosiy vazifa.
- Eng muhim 4 signal doim ko‘rinadi; 20 ta formulali xulosa talab bo‘lganda ochiladi.
- Qo‘lda kiritiladigan suv ssenariysi sukut bo‘yicha yig‘ilgan.
- Xarita boshqaruvlari faqat dala qidirish, ekin, tavsiya va splitdan iborat.
- Dala pasporti o‘ng tomonda, suv yo‘li line charti esa qat’iy ravishda xarita ostida joylashadi.
